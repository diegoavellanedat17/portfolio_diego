Projects

almuerza.co

Frontend - Backend

Almuerza.co s a web app designed for small restaurants. The purpose is to give a dynamic menu and a portal where they can receive orders inside or outside the restaurants. Each restaurant has its own clients database. The clients also have authentication to handle the status of the orders. This system involves development in frontend using HTML, CSS, Javascript, Jquery. The authentication method was using firebase. The orders and some users' information using SQL. I also design a thermal printer that prints the order in the kitchen it works with firebase cloud functions and MQTT protocol.

Zaboo

Fullstack Engineer

Zaboo is a versatile IoT system, which offers a range of devices which allow the provision of telemetry and control information. The devices communicate through the MQTT protocol with a server developed in Python, Django, which stores the information in a non-relational database. Interaction with the user is done through a chat with Zaboo that uses AI for communication.
